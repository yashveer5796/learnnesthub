Lesson 5: Introduction to Productivity Software

This lesson introduces popular productivity software used in schools and offices: Word, Excel, and PowerPoint.

Microsoft Word

Used for creating documents, letters, and reports. Features include text formatting, tables, and images.

Microsoft Excel

Spreadsheet software used for calculations, charts, and data analysis.

Microsoft PowerPoint

Used for creating presentations with slides, text, images, and animations.
