Lesson 7: Internet Safety and Cyber Security

Introduction

The Internet is very useful, but it can also be dangerous if not used carefully. Internet safety and cyber security help us stay safe while using computers and the Internet.

What is Internet Safety?

Internet Safety means following rules and practices to protect ourselves while using the Internet.

Examples:

  • Not sharing personal information online
  • Using strong passwords
  • Avoiding unknown websites

What is Cyber Security?

Cyber Security is the protection of computers, networks, and data from online threats.

It helps keep our information safe from hackers and viruses.

Common Online Threats

  • Virus: Harmful program that damages computers
  • Malware: Software designed to harm devices
  • Phishing: Fake messages to steal personal data
  • Hacking: Unauthorized access to systems

Safe Internet Practices

  • Use strong and secret passwords
  • Do not click on unknown links
  • Install antivirus software
  • Log out after using public computers

Summary

  • Internet safety helps protect users online
  • Cyber security protects data and systems
  • Viruses and phishing are common threats
  • Safe habits keep us protected online

MCQs (Practice Questions)

  1. What does Internet safety mean?
    • A. Using Internet fast
    • B. Protecting ourselves online
    • C. Buying computers
    • D. Playing games
      • Answer: B
  2. Which of the following is a cyber threat?
    • A. Virus
    • B. Keyboard
    • C. Printer
    • D. Monitor
      • Answer: A
  3. Which is a safe Internet practice?
    • A. Sharing passwords
    • B. Clicking unknown links
    • C. Using strong passwords
    • D. Ignoring antivirus
      • Answer: C
  4. What is phishing?
    • A. Online game
    • B. Fake message to steal data
    • C. Antivirus software
    • D. Internet browser
      • Answer: B
