Lesson 4: Understanding Operating Systems and File Management

Introduction

Every computer needs a system to manage its hardware and software. This system is called an Operating System (OS). Along with the OS, proper file management helps us store and organize data efficiently.

1. Operating System (OS)

An Operating System is software that manages all the hardware and software resources of a computer.

Examples:

  • Windows
  • Linux
  • macOS
  • Android

Functions:

  • Manages computer hardware
  • Provides a user interface to interact with the computer
  • Runs and manages applications
  • Handles input and output operations

2. File Management

File Management is the process of storing, organizing, and accessing data on a computer.

Examples:

  • Creating folders and subfolders to store documents
  • Renaming, copying, or deleting files
  • Saving pictures, videos, and music in proper folders

Uses:

  • Keeps data organized and easy to find
  • Prevents loss of important files
  • Helps maintain efficient storage

Summary

  • Operating System manages hardware and software resources
  • OS provides a user interface and runs applications
  • File Management helps organize and access data efficiently
  • Proper folder structure prevents data loss

MCQs (Practice Questions)

  1. Which software manages all hardware and software resources?
    • A. Application Software
    • B. Operating System
    • C. Web Browser
    • D. Database
      • Answer: B
  2. Which of the following is an example of an operating system?
    • A. Microsoft Word
    • B. Windows
    • C. Google Chrome
    • D. VLC Media Player
      • Answer: B
  3. Which process helps organize files and folders on a computer?
    • A. Hardware management
    • B. File Management
    • C. Input processing
    • D. Internet browsing
      • Answer: B
  4. Which is NOT a function of an Operating System?
    • A. Managing hardware
    • B. Running applications
    • C. Cooking food
    • D. Handling input/output
      • Answer: C
