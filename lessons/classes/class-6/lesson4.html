Lesson 3: Basic Computer Components

In this lesson, we will learn about the main components of a computer and how hardware and software work together.

Hardware

  • CPU (Central Processing Unit): Brain of the computer
  • RAM (Memory): Temporary storage for running programs
  • Storage Devices: HDD, SSD
  • Input Devices: Keyboard, Mouse
  • Output Devices: Monitor, Printer

Software

  • System Software: Operating system, drivers
  • Application Software: Word processors, games, browsers
