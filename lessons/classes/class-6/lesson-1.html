Lesson 1: Introduction to Computers

Welcome to Lesson 1! In this lesson, we will learn about computers, their basic components, types, and how they are used in our daily lives.

What is a Computer?

A computer is an electronic device that processes data and performs tasks according to instructions. It can store, retrieve, and process information very quickly.

Types of Computers

  • Personal Computer (PC): Used by individuals for work or study.
  • Laptop: Portable computer for mobile use.
  • Server: Manages network resources and stores data for multiple users.
  • Supercomputer: Very powerful computer for complex calculations.

Basic Components

Computers have both hardware (physical parts) and software (programs) that work together to perform tasks.

  • CPU (Central Processing Unit)
  • RAM (Memory)
  • Storage (Hard Disk / SSD)
  • Input Devices (Keyboard, Mouse)
  • Output Devices (Monitor, Printer)
